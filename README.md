# NodeBB Plugin Video

NodeBB plugin play mp4 video in html5

## Installation

- npm install nodebb-plugin-video
- add `<script src="/plugins/nodebb-plugin-video/files/jwplayer.js"></script>` in ACP's Themes->Customize->Custom JS and select `Use Custom JS`
